## Descripcion

NetPractice es un proyecto introductorio de redes centrado en comprender y aplicar los fundamentos basicos de la comunicacion en red.  
El objetivo del proyecto es analizar diagramas de red que no funcionan correctamente y corregir su configuracion para que los dispositivos puedan comunicarse entre si.

A lo largo del proyecto se trabaja con la configuracion de direcciones IP, mascaras de subred y puertas de enlace por defecto, asi como routers y switches dentro de una red. Cada ejercicio requiere identificar errores de direccionamiento o enrutamiento y corregirlos aplicando principios correctos de TCP/IP.

Este proyecto pone el foco en el razonamiento sobre topologia de red, rangos de direcciones y logica de enrutamiento.

## Instrucciones

1. Descargar los archivos del proyecto.
2. Abrir el archivo `index.html`.
3. Introducir el login de 42 en la interfaz para trabajar con la configuracion personal.
4. Completar los 10 niveles disponibles corrigiendo las configuraciones incorrectas de red.
5. Para cada nivel completado correctamente, utilizar el boton **Get my config** para exportar el archivo de configuracion.
6. Colocar cada archivo de configuracion exportado en la raiz del repositorio Git.

## Detalles de la entrega

- Se deben entregar 10 archivos de configuracion.
- Cada archivo corresponde a un nivel completado.

### Uso de IA

Las herramientas de IA se utilizaron como apoyo al aprendizaje para:
- Aclarar conceptos teoricos relacionados con redes y subnetting.
- Verificar el razonamiento al diagnosticar por que una configuracion de red no funcionaba.
- Mejorar la claridad y estructura de explicaciones tecnicas.

La IA no se utilizo para resolver directamente los ejercicios ni para generar configuraciones.
